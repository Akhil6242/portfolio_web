The Idea

Developed a comprehensive dual-platform university management system addressing the critical need for secure student verification and access control. The system ensures only validated students can access university services through a robust registration-verification-access workflow, combining mobile convenience with administrative oversight.

How It Works?

Students register via Android app, with data securely stored in Firebase. Department management accesses a dedicated Admin Web Portal to validate student information and grant access permissions. Only verified students can login to the Android app and utilize university services, ensuring secure and controlled access to institutional resources.

Project Scope

Currently handles student registration, verification, and basic service access. The system is designed as a foundation for expansion into a comprehensive university campus platform. Future development will integrate all campus services including library management, fee payment, course enrollment, and facility booking into a single centralized system.

Why This Project?

Built to solve real university access control challenges while demonstrating advanced full-stack development capabilities. The project showcases Android development, web application design, database management, and system integration skills. Successfully addresses institutional needs while serving as a comprehensive portfolio demonstration of technical expertise.

Admin Portal (Web-App)

Dedicated web application for department management to control student access and manage the Android app ecosystem.

Admin Portal Features

  • Student Verification: Review and validate registration data
  • Access Control: Grant or deny Android app login permissions
  • Data Management: View, edit, and manage student information
  • Bulk Operations: Manage multiple student records efficiently
  • Secure Authentication: Firebase-based admin login system
